## Overview
This repository provides a beginner-friendly Python scaffold with tooling for testing, linting, formatting, CI, and optional future HuggingFace / ML integration.

## Getting started

1. Create a virtual environment:
   python -m venv .venv
   source .venv/bin/activate

2. Install dependencies:
   pip install -r requirements.txt

3. Run tests:
   pytest -q

## Project layout
- src/socionics_research: package source
- tests/: test suite
- docs/: documentation
- .github/: CI and templates

See PROJECT_KICKSTART.md for implementation notes and next steps.
